# Chrome 抓单扩展 · 安装

店小秘抓单扩展（Chrome MV3）无法打进 exe，需在浏览器单独加载。本期多门店统一用「加载已解压」。

## 安装步骤（每台机器一次）

1. Chrome 地址栏输入 `chrome://extensions` 回车。
2. 打开右上角 **开发者模式**。
3. 点 **加载已解压的扩展程序**，选中本包内的 `extension-dist\` 目录。
4. 列表里出现该扩展、状态为「已启用」即成功。
5. 打开店小秘订单页，按扩展说明抓单；抓到的单会经本机后台服务（`127.0.0.1:8770`）写入收件夹，
   开花桌面会自动载入。

> 前提：总启动器已打开且状态显示「:8770 运行中」（否则扩展抓到的单无处可投）。

## 注意

- **开发者模式扩展**：Chrome 每次启动可能弹「要停用开发者模式扩展吗」，选「保留」即可；不要点停用。
- 扩展只在你打开的店小秘标签页工作，不会后台访问其它网站。

## 后续：切换到「未上架 Chrome 应用商店」（计划）

注册 Google 开发者后，可把同一份扩展以**未上架（Unlisted）**方式上传商店：
之后每台机器只需点一个安装链接即可，**自动更新、无需开发者模式、不再弹警告**。
扩展代码不变，只是换分发渠道——届时本文档会更新为商店安装链接。
